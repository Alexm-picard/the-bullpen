## Known edge cases

- **Cache stampede**: Caffeine with `refresh-after-write` prevents on-cold-cache thundering herd. Configure once.
- **Mode = `'shadow'` with traffic_pct > 0**: should be rejected (in shadow mode, no traffic to challenger). Validate in service.
- **First-ever model with no routing row**: `getRouting` returns the only registered champion's config implicitly. Or eagerly insert a routing row on first promotion to champion.
