# 3a.5 — Training snapshot storage + B2 archive + reconciliation

> Owning phase: `phase-3-ml-wrapper/3a-registry` · Estimated effort: `5–7 h` · Authored: 2026-05-10

---

## Scope boundaries

**IN scope**:
- Storage layout for `models/<model_name>/<version>/{model.onnx,calibrator.json,metadata.json,feature_pipeline.json,training_data.parquet}`
- Local retention: keep last 5 versions per `model_name`; archive older to B2
- B2 archive prefix `models-archive/<model_name>/<version>/`
- Restore-from-B2 helper (`RegistryService.restoreVersion(versionId)`) downloads artifacts back to local
- Reconciliation job: weekly worker job that lists `prediction_log.model_version_id` and warns on any IDs not in `model_versions` (Risk Register G2)

**OUT of scope**:
- The B2 backup pipeline at the bucket level — already done in 0.10 with separate `daily/weekly/monthly` prefixes
- Model-deletion semantics — never delete; only archive

---

## Objectives

1. New model registration enforces local artifact paths under `/var/lib/thebullpen/models/`.
2. After registration, a local-retention sweep moves older versions to B2 and updates `artifact_path` to point to a B2 URI.
3. Restoring an archived version re-downloads from B2 and updates the path.
4. Reconciliation job runs weekly and pages on cross-DB integrity violations.

## Step-by-step implementation tasks

1. Author `SnapshotStorage`:
   ```java
   public Path placeArtifacts(String modelName, String version, Path tempDir) {
       Path target = Path.of("/var/lib/thebullpen/models", modelName, version);
       Files.createDirectories(target);
       for (var f : List.of("model.onnx","calibrator.json","metadata.json","feature_pipeline.json","training_data.parquet")) {
         Files.copy(tempDir.resolve(f), target.resolve(f), StandardCopyOption.REPLACE_EXISTING);
       }
       return target;
   }

   public void enforceRetention(String modelName) {
       var versions = repo.findByName(modelName).stream()
         .filter(v -> v.stage() != Stage.ARCHIVED)
         .sorted(Comparator.comparing(ModelVersion::trainedAt).reversed())
         .toList();
       var toArchive = versions.size() > 5 ? versions.subList(5, versions.size()) : List.<ModelVersion>of();
       for (var mv : toArchive) {
         b2.upload(mv.artifactPath(), b2Path(mv));
         repo.updateArtifactPath(mv.id(), b2Path(mv));
         FileUtils.deleteDirectory(Path.of(mv.artifactPath()).toFile());
       }
   }
   ```
2. Author `B2ArchiveClient`:
   ```java
   public void upload(String localPath, String b2Path) { runRclone("copy", localPath, "b2:" + bucket + "/" + b2Path); }
   public void download(String b2Path, String localPath) { runRclone("copy", "b2:" + bucket + "/" + b2Path, localPath); }
   ```
   Use `ProcessBuilder("rclone", ...)`. Validate exit code = 0.
3. Author `ReconciliationJob`:
   ```java
   @Component
   @Profile("worker")
   public class ReconciliationJob {
     @Scheduled(cron = "0 0 4 * * SUN", zone = "America/New_York")
     public void run() {
       List<Long> known = repo.findAllIds();
       List<Long> seen = clickhouse.query("SELECT DISTINCT model_version_id FROM prediction_log WHERE request_at > now() - INTERVAL 7 DAY", Long.class);
       List<Long> orphans = seen.stream().filter(id -> !known.contains(id)).toList();
       if (!orphans.isEmpty()) discord.send(NOTICE, "Orphan model_version_id in prediction_log", Map.of("ids", orphans.toString()));
     }
   }
   ```
4. Author `restoreVersion(versionId)`:
   - Looks up `artifact_path` (B2 URI).
   - Downloads to `/var/lib/thebullpen/models/<name>/<version>/`.
   - Updates `artifact_path` to local path.
5. Author `SnapshotStorageIT`: place artifacts → enforce retention with 6 versions → 1 ends up archived.
6. Document in `ops/runbooks/registry-snapshot-recovery.md`.

## Known edge cases

- **Champion / shadow archived locally**: never. Retention only considers `stage IN ('candidate','archived')` for archival; champion + shadow stay local always.
- **B2 upload failure mid-archive**: leave local files intact; do NOT update `artifact_path`. Retry on next sweep.
- **`rclone` not on PATH**: fail at startup with a clear error.
- **Disk-pressure prioritization**: if local disk > 80% full, archive aggressively (down to last 3 versions). Surface as a NOTICE.
- **Model name with slashes**: forbid in `register` validator. `model_name` is `^[a-z0-9_]+$`.
