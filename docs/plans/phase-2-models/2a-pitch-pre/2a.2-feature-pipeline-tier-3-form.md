## Known edge cases

- **Mid-season pitcher debut**: `days_since_last_appearance` is undefined for first appearance â€” use `NULL`, not 0.
- **Doubleheaders**: same `pitcher_id` + same `game_date` in two `game_id`s. Counters must recognize game boundaries via `(pitcher_id, game_id)` not just `(pitcher_id, game_date)`.
- **In-game counter at pitch 1**: should be 0, not NULL.
- **Cross-season boundary**: `season_to_date` resets at the season boundary; pitchers traded mid-season keep one rolling history (not split per team in v1).
- **`description` filter**: only count pitches with `description != 'unknown'` for rate calculations.
