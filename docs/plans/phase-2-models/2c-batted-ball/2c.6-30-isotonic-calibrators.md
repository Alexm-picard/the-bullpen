## Step-by-step implementation tasks

1. Author `calibration.py`:
   ```python
   def fit_per_park_calibrators(model, val_loader, n_parks=30, n_outcomes=5):
       all_logits, all_labels = collect_predictions(model, val_loader)  # shape (N, 30, 5)
       per_park_calibrators = []
       for p in range(n_parks):
           park_logits = all_logits[:, p, :]
           park_labels = all_labels[:, p, :]   # one-hot via argmax of retrodicted distribution
           per_class = []
           for c in range(n_outcomes):
               iso = IsotonicRegression(out_of_bounds='clip').fit(park_logits[:, c], park_labels[:, c])
               per_class.append(iso)
           per_park_calibrators.append(per_class)
       return per_park_calibrators

   def transform(per_park_calibrators, raw_proba):  # (B, 30, 5)
       calibrated = np.empty_like(raw_proba)
       for p in range(30):
           for c in range(5):
               calibrated[:, p, c] = per_park_calibrators[p][c].transform(raw_proba[:, p, c])
       calibrated = calibrated / calibrated.sum(axis=-1, keepdims=True)  # renormalize
       return calibrated

   def to_json(per_park_calibrators) -> dict: ...
   def from_json(d) -> list[list[IsotonicRegression]]: ...
   ```
2. Persist `calibrator.json` next to `model.onnx` for `batted_ball/v1`.
3. Generate 30 per-park reliability-diagram PNGs and add them to eval artifact under `eval/reliability_diagrams_per_park/`.
4. Author `test_calibration.py`:
   - Synthetic MLP outputs with intentional miscalibration on park 5 → after calibration, ECE drops on park 5 specifically.
   - JSON round-trip preserves predictions to within `1e-9`.

## Known edge cases

- **Low-data parks**: Oakland's 2024 attendance / few HRs at COL → small calibration sample. Document any park with N < 200 BIPs and accept higher ECE there.
- **Calibrator monotonicity break under renormalization**: same trade-off as 2a.5; renormalization may slightly violate per-class monotonicity. Acceptable.
- **Java port**: Java needs the same 30×5 structure. Phase 2c serving leaf (in 2c.7's wake) will load this JSON identically to 2a.8's `IsotonicCalibratorJava`.
- **`out_of_bounds='clip'`**: necessary to handle extreme MLP outputs that fall outside the training distribution.
