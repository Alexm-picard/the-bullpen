# Design explorations

Static design mockups captured during UI exploration. They are **historical artifacts**,
not a live spec - the shipped UI is the source of truth, and the locked visual identity is
documented in `docs/decisions.md` (decision [160], the broadcast-graphics redesign) and the
Mantine/Tailwind theme tokens under `frontend/src/design/`.

Kept for provenance (the design reasoning behind the current screens) and relocated here from
the repo root so they stop cluttering the top level.

Rough groupings by filename prefix:

- `01`-`14` - the original screen set and the first parks/home redesign passes.
- `22`-`31` - token foundation + the matchup-feature exploration (judge card, pitch mix, spray, heatmap).
- `32`-`49` - the responsive desktop/tablet/mobile set per screen (home, ops, parks, games, about, game-detail).

These do not render in the app and are not referenced by any code; treat them as a sketchbook.
